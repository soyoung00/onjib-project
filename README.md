# 소개🐶🐱

<p>
우리 사이트는 유기동물, 특히 개와 고양이를 위한 <b>입양, 파양, 입소, 봉사, 후원</b> 서비스를 제공하는 플랫폼입니다.<br>
누구나 쉽게 참여하고 도움을 줄 수 있도록 <b>직관적인 UI와 팝업폼 기반 기능</b>을 제공합니다.<br>
또한, <b>입양 후기 작성과 입양 혜택 안내</b>, <b>슬라이드·API 연동</b> 등을 통해 반려동물과 사람이 함께 행복할 수 있는 환경을 만듭니다.
</p>

---------
## 1. 주제🐾
유기동물(개, 고양이)을 위한 다양한 서비스를 제공하여, 사람들이 쉽게 참여하고 반려동물과 행복하게 공존할 수 있는 **감성적이고 신뢰감 있는 플랫폼**을 만드는 것이 목표입니다.

## 2. 목표🎯
- 유기동물에게 새로운 가족을 찾아주는 **입양/파양&입소 지원**  
- 봉사, 후원 등 참여를 통해 **보호소 운영 및 동물 복지 향상**  
- 사용자 친화적인 UI/UX로 **입양·봉사·후원 경험을 간편하게 제공**  

## 3. 핵심 기능📱
- **입양 문의**: 입양가능한 동물의 정보확인 가능 및 팝업 폼으로 간편하게 문의 제출 
- **파양&입소 문의**: 팝업 폼으로 간편하게 문의 제출  
- **입양 후기**: 입양한 반려동물 경험 공유 및 댓글기능 및 후기작성 가능
- **후원 신청**: 다양한 방법으로 유기동물 후원 가능  
- **봉사 신청**: 보호소 봉사 활동 참여 신청  
- **입양 혜택 안내**: 카테고리별 입양 혜택 정보 제공  
- **외부 API 연동**: 동물 정보 및 공지사항 실시간 제공  
- **반응형 지원**: 모바일(~768px) 및 태블릿(~1024px) 환경 최적화  

## 4. 주요 기술💻
- **Front-End**: HTML, CSS, JavaScript  
- **데이터 관리**: JSON, 외부 API 연동

## ⚡ 주요 기능

- 📝 **입양문의**  
  입양가능한 반려동물 리스트 확인가능. 간단한 입양절차 안내. 팝업 폼으로 간편하게 입양문의를 제출 가능.

- 🏠 **입소문의**  
  간단한 입소절차 안내. 유기동물 입소 관련 문의를 팝업 폼으로 제출 가능.

- ✍️ **입양후기**  
  입양한 반려동물의 경험을 공유하여 다른 사용자들에게 정보 제공 및 신뢰 형성. 후기 작성도 가능하며 각 게시물에 댓글 남기기가 가능.

- 🎁 **입양혜택**  
  카테고리별 입양 혜택 정보 제공하여 입양 전 참고 가능.

- 💖 **후원하기**  
  일시후원과 정기후원 선택 후 제출 가능. 보호소 운영 지원.

- 🤝 **봉사하기**  
  보호소 봉사 활동 참여 신청 가능. 

- 📍 **오시는길**  
  보호소 위치 안내 및 지도 제공.

## 💥 트러블슈팅

### 📌 1. 데이터가 하나의 필드에 통합되어 있어 필요한 값 분리가 어려운 문제
- **원인**  
  여러 정보가 하나의 문자열 필드에 결합되어 있어, 단순한 값 추출 방식으로는 필요한 데이터를 분리하기 어려웠음.  
- **해결**  
  문자열 내 구분자를 기준으로 `split()`과 `substring()`을 활용해 필요한 데이터를 각각 분리하도록 처리함.  

### 📌 2. JSON 데이터를 그대로 출력해 페이지 이동 시에도 동일한 순서로 렌더링되는 문제
- **원인**  
  기본 배열 순서가 그대로 유지되어, 페이지 이동 시 사용자가 순서 변경 여부를 직관적으로 확인할 수 없었음.  
  `Math.random()` 사용 시 새로고침마다 순서가 달라져 일관성이 없었음.  
- **해결**  
  Seed 기반 난수 함수를 구현하여 고정된 랜덤 정렬을 적용함.  
  페이지 이동 시 순서는 달라 보이지만 새로고침해도 동일한 순서를 유지하도록 개선함.  

### 📌 3. 입양 상세 페이지에서 새로고침 또는 직접 URL 접근 시 데이터가 표시되지 않는 문제
- **원인**  
  상세 페이지가 목록 페이지에서 전달된 데이터에 의존하도록 설계되어, 새로 로드될 경우 참조할 데이터가 존재하지 않았음.  
- **해결**  
  `localStorage`에 선택된 데이터를 저장하고, 상세 페이지 로드 시 해당 데이터를 불러와 화면에 출력하도록 구현함.
